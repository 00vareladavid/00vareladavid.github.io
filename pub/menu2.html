A bit more highlighting

Extension of highlighting for pkg an shell mode in Julia:

(v1.4) pkg> add JuDoc
shell> blah
julia> 1+1
(Sandbox) pkg> resolve

you can tune the colouring etc via the

  • hljs-meta (for julia>)

  • hljs-metas (for shell>)

  • hljs-metap (for ...pkg>)
